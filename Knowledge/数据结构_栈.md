### 一、知识点


C++中的Stack（堆栈）是一个容器类的改编，其最重要的特性为先进后出（FILO），基础数据结构之一。

要在C++中调用stack需要声明
```
#include<stack>
```

栈最重要的概念是栈顶

Stack成员函数包括：

```
empty() 堆栈为空返回true

pop() 移除栈顶元素

push() 在栈顶增加一个元素

size() 返回栈中元素数目

top() 获取并返回栈顶元素

----------不常用函数----------
swap() 交换两个栈的元素

```

stack被限制了插入、删除、增加，操作位置都是top，即栈顶位置。

该特性也可以理解为除了栈顶以外的元素都是不可见的，==即不可以被遍历，没有迭代器，不支持**随机访问**==。


### 二、操作

1、初始化：

```
stack <int> s;
```

2、赋值：

```
由于栈的特性只能从栈顶压入，如果存多个则需要用循环

s.push()
```
3、返回值：

```
#返回是否为空，空则返回true
s.empty()

#返回顶部元素
s.top()

#返回栈内数量
s.size()
```
